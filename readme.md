# 🔋 精力管理可视化 (Energy Manager)

一款将“个人精力”进行游戏化数值管理的 Chrome 浏览器扩展。通过记录日常行为（睡眠、饮食、运动、番茄钟）来动态计算和可视化当前的精力状态，激励自己合理规划作息，保持高效与专注。

## 💡 产品设计理念
* **核心思想：** 可视化精力管理。将人体的精力抽象为类似游戏角色中的“体力条（Stamina）”或“生命值（HP）”。
* **行为正负反馈：** 好的生活习惯（按时吃饭、睡够 8 小时、保持专注）会恢复精力或提升精力上限；而熬夜、错过饭点、虚度光阴则会加速精力流失或降低精力上限。
* **强干预机制：** 当精力值降至临界点时，强制打断工作流程，提醒用户休息，避免过度透支。

## ✨ 当前功能 (MVP 版本)

### 1. 实时精力仪表盘
* 浏览器主界面提供直观的**精力进度条**（支持红蓝状态变色预警）。
* 随时间流逝自动扣减精力，扣减速率受日常打卡行为影响。
* 当精力值低于 20 时，会自动弹出 Chrome 新标签页进行强提醒。

### 2. 沉浸式番茄钟
* 默认 25 分钟倒计时，配合**环形进度条**和状态蒙层（支持播放/暂停切换）。
* 提供快捷“刷新”按钮，支持快速打断并重置番茄钟。
* 番茄钟结束后弹出结算页，支持对该时段的“专注度”进行打分（0-100%）。

### 3. 每日任务矩阵
* 紧凑的 2x4 任务打卡面板，包含：睡眠、运动、早中晚三餐时间录入，以及 3 次无时间限制的喝水打卡。
* 防误触设计的按钮与状态锁定机制，打卡后即时生成日志。

### 4. 数据统计看板
* **折线图趋势：** 集成 Chart.js，直观展示最近 7 天的 4 项核心指标变化（精力上限、精力消耗值、番茄总数、完美番茄数），支持图例点击筛选。
* **实时日志流：** 记录每一次打卡、番茄钟完成及状态变更，支持容器内无限下拉滚动查看。

## 🧮 精力计算与结算规则

每天早上 **8:00** 触发跨天结算与数据重置。

### 1. 精力上限 (Max Energy)
* **初始值：** 默认为 63。
* **每日变更逻辑（跨天结算时触发，最低跌至 10）：**
  * **日常任务：** 昨天完成所有每日任务（睡够8h + 三餐按时 + 运动30min + 喝水3次），上限 **+1**；否则 **-1**。
  * **番茄任务：** 昨天完成的“完美番茄钟（100%专注度）”每满 4 个，上限 **+1**；若全天完成的番茄钟总数 < 4 个，上限 **-1**。

### 2. 精力消耗 (Decay)
* **基础速率：** 每天早上 8 点至晚上 12 点，精力呈线性自动扣减，默认速率为 **4 点 / 小时**。
* **边缘保护：** 精力值最低扣减至 **5**，到达后不再自然下降。
* **惩罚机制：** 如果未在规定时间内打卡用餐（9:00 前无早饭 / 13:00 前无午饭 / 19:00 前无晚饭），扣减速率将变为 **1.5 倍**（即 6 点 / 小时），直到完成对应打卡。

### 3. 精力恢复 (Recovery)
* **初始状态：** 每天早上 8 点结算后，当日精力默认初始化为上限的 **80%**。
* **睡眠补偿：** 录入睡眠时间且 >= 8 小时，瞬间补齐剩余 **20%** 的精力值（已随时间流逝扣减的部分不补）。
* **按时吃饭：** 在规定时间内打卡（早饭 <= 08:00，午饭 <= 12:00，晚饭 <= 18:00），每次瞬间恢复 **5 点** 精力。
* **完美专注：** 完成一个专注度为 100% 的番茄钟，精力值 **+1**。

## 🛠️ 本地安装指南
1. 将本项目所有代码文件（HTML, JS, JSON 以及 Chart.js）放置于同一文件夹内。
2. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
3. 开启页面右上角的 **“开发者模式”**。
4. 点击左上角 **“加载已解压的扩展程序”**，选择该文件夹即可运行。
5. 建议将插件固定在浏览器工具栏，以便随时查看和打卡。
