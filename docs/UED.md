# 精力管理可视化 - 交互设计文档 (UED)

## 1. 页面/组件清单

### 1.1 Chrome 扩展端

| 页面 | 组件 | 说明 |
|------|------|------|
| 弹窗主页 | PopupApp → MainDashboard | 精力条 + 番茄钟 + 打卡任务 |
| 弹窗-规则 | PopupApp → RulesPage | 精力规则 + 同步规则说明 |
| 弹窗-统计 | PopupApp → StatsPage | 趋势图 + 日志 |
| 弹窗-设置 | PopupApp → SettingsPage | 参数配置 + 任务管理 |
| 侧边导航 | MenuPanel | 滑出式侧边栏 |
| 同步面板 | SyncPanel | 同步按钮 + 删除数据 (弹窗底部) |
| 全屏提醒 | FinishApp | 番茄完成评分 / 低精力警告 |

### 1.2 Web 端

| 页面 | 组件 | 说明 |
|------|------|------|
| 主页 | WebApp → MainDashboard (flat) | 同扩展，无卡片边框 |
| 规则/统计/设置 | 同扩展复用 | 完全相同的组件 |
| 登录面板 | AuthPanel | Google + 邮箱密码登录 + 同步 (底部) |
| 全屏覆盖 | FinishOverlay | 低精力警告 / 番茄完成评分（当前页内全屏覆盖，点击关闭回到主页面） |

## 2. 核心用户流程

### 2.1 每日使用流程

```
用户打开扩展
     │
     ▼
┌──────────────┐
│  主面板       │
│ ┌──────────┐ │
│ │ 精力条   │ │  ← 查看当前精力
│ │ 72% ████ │ │
│ └──────────┘ │
│ ┌──────────┐ │
│ │ 番茄钟   │ │  ← 开始专注
│ │  ▶ 25:00 │ │
│ └──────────┘ │
│ ┌──────────┐ │
│ │ 打卡区   │ │  ← 记录任务
│ │ 🏃+  💧+ │ │
│ └──────────┘ │
│ ┌──────────┐ │
│ │ 同步状态 │ │
│ └──────────┘ │
└──────────────┘
```

### 2.2 番茄钟流程

```
点击圆形区域 → 切换 开始/停止 (status: ongoing | idle)
     │
     ▼
倒计时 25:00 → 24:59 → ... → 0:00
     │
     ▼
┌────────────────┐    ┌──────────────────┐
│ 连续 <= 3 个?   │─否→│ 强制休息提醒     │
│                │    │ "大脑需要充电"    │
└───────┬────────┘    │ (红色主题)        │
        │是           └──────────────────┘
        ▼
┌────────────────┐
│ 专注评分页面   │
│                │
│  ◀━━━●━━━━▶   │  ← 滑动评分 0-100%
│    当前: 85%   │
│                │
│  [提交]        │
└────────────────┘
        │
        ▼
  记录日志 → 更新统计
  100% = 完美番茄 🌟
```

### 2.3 打卡任务流程

```
┌─ Counter 类型 (如饮水) ─────────┐
│                                  │
│  💧 饮水  [2/5]  [+]           │  ← 点击 [+] 计数+1
│                                  │
│  达到 maxCount 后按钮置灰        │
└──────────────────────────────────┘

┌─ Boolean 类型 (如拉伸) ─────────┐
│                                  │
│  🧘 拉伸  [ ] → [✓]            │  ← 点击切换完成状态
│                                  │
└──────────────────────────────────┘

┌─ Number 类型 (如睡眠) ──────────┐
│                                  │
│  😴 睡眠  [___] 小时            │  ← 输入数值，blur 或回车提交
│                                  │
└──────────────────────────────────┘

打卡后:
  → 根据 healLevel 恢复精力
  → 精力条动画更新
  → 追加日志记录
```

### 2.4 日期翻转流程 (后台自动)

```
每分钟 tick
     │
     ▼
当前时间 ≥ 8:00 AM 且 logicalDate ≠ 今天?
     │
     ├── 否 → 正常衰减 + 番茄倒计时
     │         (餐食惩罚: 10:00后未吃1餐 / 14:00后未吃2餐
     │          / 19:00后未吃3餐 → 衰减×penaltyMultiplier)
     │
     └── 是 → 日期翻转
              │
              ▼
         当天有日志记录?
              │
         ┌────┴────┐
         │         │
        有日志   无日志
         │      (节假日豁免
         │       不扣精力上限)
         │         │
         ▼         ▼
    判定昨日表现  跳过判定
         │
    ┌────┴────┐
    │         │
完美一天?  糟糕一天?
(计入完美   (0完美番茄
一天的任务  + 运动<30m
全部完成    + 睡眠<6h)
+ 4完美番茄)  │
    │         │
maxEnergy+1 maxEnergy-1
    │         │
    └────┬────┘
         │
         ▼
    记录 StatsSnapshot
         │
         ▼
    重置: tasks/pomodoro/energy
    energy = maxEnergy (按睡眠缩放)
```

### 2.5 跨设备同步流程

```
用户登录 (Google / 邮箱密码)
         │
         ▼
    ┌─ 扩展端 ─┐     ┌─ Web 端 ──┐
    │ chrome.  │     │ signIn    │
    │ identity │     │ WithPopup │
    │ / Email  │     │ / Email   │
    └────┬─────┘     └─────┬─────┘
         │                 │
         └────────┬────────┘
                  │
                  ▼
           登录成功 → 自动 sync() 双向合并
                  │
                  ▼
         ┌─ 合并规则 ──────────────┐
         │ 日志: timestamp|action  │
         │       去重 + 排序       │
         │ 任务: 取完成方          │
         │ 状态: 各字段取保守值    │
         │ 番茄钟: 原子同步        │
         └───────────┬─────────────┘
                     │
                     ▼
          合并结果同时写入本地和云端
                     │
                     ▼
          启动自动同步 (每 60s)
                     │
                     ▼
          用户操作 → 本地更新 → 60s 后双向同步
                     │
                     ▼
          其他设备打开 → sync() → 获取最新数据
```

## 6. 设计规范

### 6.1 颜色系统

| 用途 | 颜色 | 说明 |
|------|------|------|
| 精力高 (>60%) | `#22c55e` (green-500) | 健康状态 |
| 精力中 (30-60%) | `#f97316` (orange-500) | 需注意 |
| 精力低 (<30%) | `#ef4444` (red-500) | 危险状态 |
| 背景 | `#f9fafb` (gray-50) | 浅灰底色 |
| 卡片背景 | `#ffffff` (white) | 扩展端卡片 |
| 文字主色 | `#111827` (gray-900) | 标题/数值 |
| 文字次色 | `#6b7280` (gray-500) | 说明/标签 |
| 强调色 | `#3b82f6` (blue-500) | 按钮/链接 |
| 番茄色 | `#ef4444` (red-500) | 番茄钟主题 |

### 6.2 字体

- 系统默认无衬线字体栈 (Tailwind `font-sans`)
- 数值显示使用等宽字体 (`font-mono`)
- 标题: `text-lg font-bold`
- 正文: `text-sm`
- 标签: `text-xs text-gray-500`

### 6.3 间距规范

| 元素 | 间距 |
|------|------|
| 页面内边距 | `p-4` (16px) |
| 卡片间距 | `space-y-4` (16px) |
| 卡片内边距 | `p-3` (12px) |
| 任务网格间距 | `gap-2` (8px) |
| 按钮内边距 | `px-3 py-1` |
| 图标与文字 | `gap-1` (4px) |

### 6.4 交互规范

| 交互 | 说明 |
|------|------|
| 按钮点击 | Tailwind hover/active 状态 |
| 任务打卡 | 即时反馈，精力条动画更新 |
| 侧边栏 | 左侧滑出，半透明遮罩 |
| 番茄钟 | 圆形 SVG 进度条 |
| 输入框 | blur 或 Enter 触发提交 |
| 加载/同步 | 按钮文字变化提示状态 |

### 6.5 响应式

| 端 | 宽度 | 处理 |
|------|------|------|
| Chrome 弹窗 | 400px 固定 | 卡片布局，2 列任务网格 |
| Web 端 | max-width 容器 | 居中显示，平铺布局 |
| 移动浏览器 | 自适应 | Tailwind 响应式工具类 |
